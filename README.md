### Breezyair (`breezyair` branch)
A Neo-Brutalist HVAC service website built for **Breezyair**, Bengaluru's trusted AC repair and maintenance company.
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Deployment**: Vercel

To work on Breezyair:
```bash
git checkout breezyair
cd web
npm install
npm run dev
```
